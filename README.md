# Assets Publics (React / Vite)

Placez vos fichiers d'images et documents ici (par exemple `notif.pdf` et `amgabon.png`).
Vite copiera automatiquement tous les fichiers de ce dossier vers la racine du dossier de build `dist/` lors de la compilation (`npm run build`).

Ainsi :
- `public/notif.pdf` sera accessible via `src="notif.pdf"` ou via les liens de document.
- `public/amgabon.png` sera accessible via `src="amgabon.png"`.
